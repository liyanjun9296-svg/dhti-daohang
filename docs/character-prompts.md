## 生图参数建议

| 参数 | 推荐值 |
|------|--------|
| 风格 | 扁平卡通 / Flat Illustration |
| 尺寸 | 1:1 正方形（400×400 或 800×800） |
| 背景 | 纯色或简单渐变（每个角色用独立主题色） |
| 文字 | 无文字（文字由前端叠加） |
| 工具 | Midjourney v6 / DALL-E 3 / 即梦AI |

> 提示：生成后统一命名为 `{CODE}.png`（全大写），存入 `frontend/src/assets/types/` 目录。
